### How to Test the Application

1.  Visit the [React Application link](https://yketu-csc3916-react.onrender.com).
2.  **Sign up** for a new account or **log in** with an existing one.
3.  Browse the top-rated movies on the main screen.
4.  Click on any movie to see its details, actors, and reviews.
5.  Use the **search bar** to find movies by title or actor name.
6.  On a movie detail page, write a new review by selecting a rating and entering a comment.
